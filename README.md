# CS202
Multi person Git Test

## Steps

1. Clone the Repository to your local device and make sure you can see this README file

2. Add a File to the Repository (Titled your first and last name in upper camel case like JohnDoe.txt). Then pull and push this to the repository.

3. Conflict! We will try to add out name to Names.txt, but they will be different, and one of us will have to resolve the merge conflict

4. Create a branch where you create a new directory and add your named file from step 2 into your directory and merge the branch in

5. Make sure we are in the same Canvas group for this assignment, and turn ina screenshot of our git log terminal and GitHub repo
